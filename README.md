# Suwate26.com - Domain For Sale Website

A professional, modern landing page for selling the premium domain **Suwate26.com**.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Contact Form**: Built-in inquiry form for potential buyers
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Easy to Customize**: Simple HTML, CSS, and JavaScript structure

## Structure

```
domain-website/
├── index.html      # Main HTML file
├── styles.css      # All styling
├── script.js       # JavaScript functionality
└── README.md       # Documentation
```

## How to Use

1. Open `index.html` in a web browser to view the website
2. Customize the content as needed
3. Deploy to your web hosting service

## Customization

### Update Contact Information
Edit the form submission logic in `script.js` to integrate with your email service or backend API.

### Change Colors
Modify the color scheme in `styles.css`:
- Primary color: `#667eea`
- Secondary color: `#764ba2`

### Add Backend Integration
To receive actual form submissions, integrate with:
- FormSpree
- Netlify Forms
- Your own backend API
- Email service (SendGrid, Mailgun, etc.)
